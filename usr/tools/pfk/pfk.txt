Programming for Kids
--------------------

Virtual Machine
---------------

- 4 32-bit general-purpose registers EAX, EBX, ECX, EDX
- 32-bit program counter
- 32-bit frame pointer
- 32-bit stack pointer
- 32-bit machine status word

Programming Interface
---------------------

- initial support for programming in custom assembly
  - assembler + linker + bytecode interpreter
- later to be extended to be programmable in other languages
  - Forth
  - BASIC
  - Pascal
  - ...
