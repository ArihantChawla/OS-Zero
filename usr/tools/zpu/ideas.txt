Zero Processor Unit Ideas
-------------------------

Rational Numbers
----------------
- pairs of 32-bit nominator and denominator
- assembly immediate: $(n, d)
- C syntax: 0xNNNNNNNN_DDDDDDDD

Special Values
--------------
{ 0, 0 }	not a number (NaN)
{ INT_MAX, 0 }	positive infinity
{ INT_MIN, 0 }	negative infinity
