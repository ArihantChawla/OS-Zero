debugger interface
------------------

	-----------------------------------------
	|	|	|		|	|
	| WIN1	| WIN2	| WIN3		| WIN4	|
	| 	| 	| 		| 	|
	-----------------------------------------

WIN1
----
- REG; ALU, STK, FPU, VEC
  - registers for ALU, expression stack, FPU, and SIMD

WIN2
----
- SYM; auto, global, dynamic
  - automatic, global, and dynamic symbols (with values)

WIN3
----
- EXEC; disassembly, source

WIN4
----
- CALL; call stack, system calls
