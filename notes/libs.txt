Zero Foundation Libraries
-------------------------
- libzero
  - library shared between kernel and the c library
- libc
  - ISO C
  - netinet (sockets etc.)
- libvt
  - VT100 and up terminal emulation
- libtext
  - UTF-8, ISO 8859, ISO10646/Unicode (both 16- and 31-bit)
- libimg
  - tga, raw, png, jpeg, gif
- libaud
  - wav, flac, ape, wma, mp3
- libvid
  - dvd, bluray
- libsnd
  - seq, mixer, synth, patch, sample
  - waveforms with functions
  - 44100 KHz 16-bit WAV samples
  - software synthesizers
    - VST?
